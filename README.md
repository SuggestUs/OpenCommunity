## How to run locally

- Fork the repository
- Clone the repository
  ```bash
  git clone https://github.com/your-github-username/OpenCommunity
  ```
- Navigate to the project directory
  ```bash
  cd OpenCommunity
  ```
- Install dependencies
  ```bash
  pnpm i
  ```
- Run the project
  ```bash
  pnpm dev
  ```
